Step 1 — Bot Token

Your bot token is a secret password that lets the code control your bot.

How to get your token:
  1. Go to discord.com/developers/applications
  2. Click your app (or New Application if you haven't made one)
  3. Click Bot in the left sidebar
  4. Click Reset Token → copy the long string that appears
Keep this secret — never share it publicly.
Token looks too short. Double-check you copied the full string.

Step 2 — Application (Client) ID

This is your bot's unique ID number — different from the token.

How to find it:
  1. On the Developer Portal, click your app
  2. Click OAuth2 in the left sidebar
  3. Copy the Client ID — it's a long number
Numbers only — usually 17–19 digits long.
Client ID should be a long number. Check you copied it correctly.

Step 3 — Server (Guild) ID

This tells the bot which Discord server to register its commands on.

How to find it:
  1. In Discord, open User Settings → Advanced and turn on Developer Mode
  2. Right-click your server name in the left sidebar
  3. Click Copy Server ID
Numbers only — same length as the Client ID.
Guild ID should be a long number. Check you copied it correctly.

Step 4 — Deploy Mode

Choose how commands get registered. You can always change this later in the .env file.

Guild registers commands only to your chosen server instantly. Use Global only if you plan to add the bot to many servers.
