What is Sigil

Everything your server needs to look professional.

Sigil combines a Discord slash-command bot with a visual web GUI. Use the browser tools to design and preview assets live, then copy the Discord command to generate them directly in your server — no uploads, no third-party apps.

How it works

Three steps to a branded server.

No installs needed for the web tools. Add the bot to your server when you're ready to run commands.

01

Design in the browser

Open the Brand Builder or Community Tools. Tweak colours, fonts, and layouts. See changes update live on canvas in real time.

02

Download or copy

Download your PNG assets directly from the browser, or copy the auto-generated Discord slash command from the Command box below each preview.

03

Run the command

Paste the copied command into your Discord server. Sigil bot renders and posts the image instantly with your exact settings.

04

Automate it

Use /sigilconfig to set up automatic welcome cards, goodbye messages, and milestone alerts — fire and forget.
