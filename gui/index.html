🎨
Brand & Identity 💻 GUI

Generate professional server icons, banners, and palette cards. Use AI to design a full brand kit from a description, or build manually with live canvas preview.

/brand kit /brand ai /brand share /icon /banner /announce /embed
  • AI-powered brand kit generation via Gemini — colors, fonts, backgrounds, and logo image from one sentence
  • Manual kit builder with 17 background styles, 6 border styles, 6 font families
  • Live canvas preview in the browser before generating in Discord
  • Shareable GUI link restores your exact settings on any device
  • /embed builder with interactive modal editor and live preview
  • Announce command for rich server announcements with optional role ping
  • Event banners and announce-banner for scheduled promotions
🛡️
Moderation 💻 GUI

Full automod suite, manual moderation actions, case logging, and audit trail. Configure every threshold from the dashboard — no commands needed.

/automod /ban /history /ticket /compare
  • Anti-spam, anti-links, anti-mentions, anti-caps, bad-words filter — each individually toggleable
  • Per-rule thresholds: spam count, mention count, caps percentage
  • Bypass role and allowed domains whitelist
  • Mod-log channel for all automod and manual actions
  • Persistent case database — every ban, kick, warn logged with case number
  • Ticket system with thread-based private support channels
  • Member history lookup with paginated case list
🎉
Community 💻 GUI

Everything that makes members feel at home — welcome cards, reaction role panels, giveaways, polls, starboard, and bump reminders.

/welcome /reactionroles /giveaway /poll /bumpreminder /autorole /starboard
  • Welcome system: custom embed, DM on join, background and color picker, live preview in dashboard
  • Reaction role panels — build multi-button panels with custom labels and colors
  • Timed giveaways with join button, winner draw, and reroll
  • Polls with multiple options, live vote tracking, and auto-close
  • Starboard: threshold-based pinning to a highlight channel
  • Bump reminder: auto-notifies when server is ready to bump again
  • Auto-roles on join or by level trigger
⭐
XP & Levels 💻 GUI

Full XP economy with configurable rate and cooldown, level-up announcements, rank cards, leaderboards, and auto role rewards on level-up.

/xprank /xpleaderboard /loyalty /levelroles
  • Passive XP gain per message with configurable rate (default 15 XP) and cooldown (default 60s)
  • Weekly XP tracking — resets every Sunday for weekly leaderboards
  • Rank card image with XP bar, level, and server rank
  • Leaderboard — top members by lifetime XP or weekly XP
  • Level-based auto-role rewards: assign roles when members reach a level
  • Loyalty command to show member join date and server tenure
📅
Scheduler & Staff 💻 GUI

Staff shift management with daily roster posts, callout system with Sylvia Ross integration, scheduled devotionals, and sermon posting.

/shift /myshift /callout /schedule /devotional /sermon
  • Staff shift board: claim and drop shifts, daily automated roster post
  • /myshift: personal shift view for each staff member
  • /callout: request shift coverage linked to Sylvia Ross scheduling system
  • Daily devotional posts with API.Bible integration — translation picker, verse lookup, random daily verse
  • Sermon command for structured ministry content posts
  • Scheduled post engine: queue any message to any channel at a future time

Integrations

YouTube upload alerts, Twitch live notifications, and custom server-specific slash commands.

/youtube /twitch /customcmd /integrations
  • YouTube: subscribe to channels, get a Discord embed when a new video drops (polls every 15 min via Data API v3)
  • Twitch: live-stream alerts posted to a configured channel when a streamer goes live
  • Custom commands: create server-specific slash commands with custom text or embed responses
  • Per-server integration settings — each guild manages its own subscriptions independently
📰
Post-Pilot ✨ New in v2.6

AI-powered social media post generation and publishing, directly from Discord. Generate captions for Facebook, Instagram, TikTok, YouTube, and more — then publish in one command.

/post /poststatus
  • /post: generate and optionally publish a social post to any combination of platforms in one command
  • /poststatus: check Post-Pilot connection health and view recent post history
  • Supports Facebook, Instagram, TikTok, YouTube, Google, and website
  • Tone selector: engaging, professional, casual, inspirational, humorous
  • Optional image URL attachment, schedule-at timestamp, and platform targeting
  • Powered by the Post-Pilot API bridge — configure via POSTPILOT_* env vars
🍳
CulinaryOS Bridge 🚀 In Development

A dedicated Discord bridge for CulinaryOS — a personal restaurant management platform in development. Syncs menus, recipes, inventory, and low-stock alerts directly into Discord channels.

/menu /recipe /inventory /lowstock /culinaryos
  • Built as a first-party bridge between CulinaryOS (a separate personal project) and Discord
  • /menu: post formatted menu embeds directly from CulinaryOS data
  • /recipe: look up and share recipes from the CulinaryOS recipe database
  • /inventory: view current stock levels synced from CulinaryOS
  • Automatic low-stock webhook alerts posted to a designated Discord channel
  • Designed to evolve alongside CulinaryOS as that platform develops
How it works

Dashboard first. Commands second.

Most settings can be configured entirely from the web dashboard. Discord slash commands are there when you prefer them — but you never have to use them.

01

Run the Setup Wizard

Open /setup and follow 5 guided steps — credentials, features, channel mapping, and slash command deploy. No terminal required.

02

Open the Dashboard

Every feature group has a dedicated panel. Pick what you want to configure and see live previews as you change settings.

03

Configure & Preview

Tweak colours, thresholds, messages, and channels. Changes render in real time so you know exactly what your server members will see.

04

Bot Runs It Live

Your Sigil bot on Discord picks up new settings immediately. Automations, alerts, and responses all reflect your dashboard configuration in real time.
